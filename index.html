編 者 前 言

《大智度論》卷四十六是龍樹菩薩釋《摩訶般若波羅蜜經》「大乘」核心義理的關鍵一卷。本卷以「乘 → 莊嚴 → 縛脫 → 衍」四個層次，依序回答了大乘菩薩道的四個根本叩問：何謂乘大乘（乘乘品）、何謂大莊嚴（無縛無脫品）、何以無大莊嚴方為大莊嚴（大莊嚴品）、摩訶衍與般若究竟是何關係（釋摩訶衍品）。

若以圖示呈現整卷邏輯：第一品「乘乘品」先解「以無所得心駕乘大乘車輿」之第一義；第二品「無縛無脫品」進一步揭示「大莊嚴」即「諸法如幻而仍負無限責任」的中觀心法；第三品「大莊嚴品」回到經文原典與龍樹釋論的雙線辯論場，把「無大莊嚴為大莊嚴」這一辯證命題層層拆解；第四品「釋摩訶衍品」最終在卷末做總攝歸結——「六波羅蜜是菩薩摩訶薩摩訶衍」，並開展十八空與四補充空之中道結構。

本閱讀本依「卷 → 品 → 章 → 節」四級層次重新編排全四品共二十四章。外層提供「總目錄」一覽全卷四品架構；各品內部保留原本「目次」呈現該品章名與節綱。每一品前置「本品要旨」、後置「承先啟後」橋接卡，引導讀者掌握品際之間的邏輯銜接。願這份彙整能幫助讀者在長篇經論的深海中，一氣貫通卷四十六「乘 · 莊 · 縛 · 衍」的義理脈絡。

卷四十六 · 第一品
乘乘品第十六
駕乘大乘 · 五重契入
本品要旨

本品承《大智度論》前文「摩訶薩」三事說的最後一事——「乘於大乘」。經文以舍利弗發問為引，富樓那一答五段，由六波羅蜜、三十七道品、假名觀、五蘊不可得，乃至神通遊化十方佛國，層層展開「乘」的內涵；論主龍樹則接續以五個「復次」逐段釋義，並以「畢竟清淨六波羅蜜」一句總攝全品宗要。所謂「乘乘」——前一「乘」為動詞，駕乘也；後一「乘」為名詞，大乘車輿也。本品要解的核心問題是：菩薩究竟以何種心態與行門，才堪稱「乘於大乘」？答案的關鍵不在車有多大，而在能乘者於一切法但見假名而不可得。

承先啟後

本品揭示「乘於大乘」之第一義——以無所得心駕乘六波羅蜜車輿。然而問題隨即翻深一層：若菩薩確以畢竟清淨之六度為乘，則此「乘」何以同時又稱為「大莊嚴」？下一品（釋無縛無脫品第十七）將承接本品「無所得即般若」的命脈，由須菩提請問「云何為菩薩摩訶薩大莊嚴」進入，由「諸法如幻、大莊嚴成」這一中觀邏輯，深入解讀六波羅蜜何以同時是無限責任的承擔，又同時是無縛無脫的解脫之行。

卷四十六 · 第二品
釋無縛無脫品第十七
諸法如幻 · 大莊嚴成
本品要旨

本品緊接前一品「乘於大乘」之後，由須菩提向佛陀重新請益：什麼是菩薩的「大莊嚴」（Mahā-vyūha）？為什麼菩薩能夠承擔起這樣浩瀚的事業而不疲、不退、不著相？佛陀的回答以六波羅蜜為主軸層層展開，從檀波羅蜜的變身化現、財法二施，到尸羅、羼提、毘梨耶、禪、般若五度的次第推演；最後以「決定誓願」破除一切數量與對象的邊界，以「諸法相畢竟空如幻」作為支撐這份無限責任的本體基礎。龍樹於【論】部以三層解碼回應：六度全攝諸功德 → 誓願不可有「我度若干、不度若干」之分別 → 以「自立如幻師」一語點出「在如幻的世間，負起無限的責任」這一核心心法。

承先啟後

本品從佛陀「諸法如幻」的本體答覆，引出「大莊嚴」即「無縛無脫」的中觀辯證。然而論場才剛拉開——「無大莊嚴何以為大莊嚴？」「自相空與無縛無脫之間如何銜接？」下一品（大莊嚴品）將回到經文原典的辯論場面，由須菩提初辯「自相空為大莊嚴」開始，經須菩提富樓那論議「無縛無脫」，再進入龍樹的雙段釋論，把「大莊嚴」這一概念在『有 / 無 / 雙非 / 雙是』四句偈上層層拆解到底。

卷四十六 · 第三品
大莊嚴品
無大莊嚴 · 為大莊嚴
本品要旨

本品標題即揭出般若系統最具辯證張力的命題——「無大莊嚴為大莊嚴」。世間所謂莊嚴，必有所莊嚴之物、能莊嚴之人，且需具備種種精進、布施、智慧等資糧；然《般若經》卻直言：菩薩若以「有所得心」修種種莊嚴行，反而非真大莊嚴；唯有徹見諸法自相空、非作非不作、無縛無脫而仍精進不退，才堪稱大莊嚴。本品在結構上分為兩大段：前半為經文，由須菩提引述佛意作初辯，引出富樓那的詰問與須菩提廣說「五蘊、三世、善惡、諸波羅蜜、十八不共法、實際法性」皆無縛無脫；後半為龍樹菩薩的釋論，逐層剖析「為何說自相空？」「為何非作非不作？」「為何無縛無解？」三大論題。

承先啟後

本品從經文原典與龍樹釋論雙線完成「大莊嚴」之中觀論證，破除一切「能莊嚴／所莊嚴」之執取。然而至此尚有一根本叩問未獲總答：六度、大莊嚴、無縛無脫——它們最終匯歸於哪個更深的範疇？下一品（釋摩訶衍品第十八）將是卷四十六的核心總攝品。由佛陀直接揭示「六波羅蜜是菩薩摩訶薩摩訶衍」這一最重要的同義關係，再展開十八空與四補充空的中道結構，把全卷的義理系統做一個徹底的歸結。

卷四十六 · 第四品
釋摩訶衍品第十八
摩訶衍即般若 · 六度十八空
本品要旨

本品全名為「釋摩訶衍品第十八」，是《大智度論》第四十六卷的核心歸結一品。所謂「摩訶衍」（Mahāyāna）即梵語「大乘」之音譯。本品所要解的對象是《摩訶般若波羅蜜經》中佛陀對須菩提的開示——「何等是菩薩摩訶薩摩訶衍」之一段。佛陀的回答出乎意料：「六波羅蜜是菩薩摩訶薩摩訶衍」，並隨即開展十八空與四補充空之深義。龍樹菩薩在論義中所要建立的核心命題有三：其一，「摩訶衍」與「般若波羅蜜」名異而義同；其二，六波羅蜜為摩訶衍之全幅體用；其三，十八空為般若慧鋒之中道結構。本品以四層論義 + 全品總持完成卷四十六的最終定盤。
